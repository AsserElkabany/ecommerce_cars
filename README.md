# E-commerce Cars Project

Welcome to the E-commerce Cars Project! This project is dedicated to the buying and selling of cars online. It aims to provide a seamless experience for users looking to purchase vehicles, as well as for sellers wanting to list their cars for sale.

## Features
- User authentication
- Advanced search filters for cars
- Detailed car listings with images
- Secure payment process
- User profiles and order tracking

## Technologies Used
- React for the frontend
- Node.js and Express for the backend
- MongoDB for the database

## Getting Started
To get a local copy up and running follow these simple steps:

1. Clone the repo
   ```sh
   git clone https://github.com/AsserElkabany/ecommerce_cars.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the application
   ```sh
   npm start
   ```
